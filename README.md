# CardMatch Monorepo

Bu repo, CardMatch uygulamasının tüm bileşenlerini içerir:

- `server/` – Node.js (Express + Socket.IO) backend ve Prisma ORM
- `mobile/` – React Native (Expo) mobil uygulama
- `admin/` – Next.js tabanlı admin paneli

Detaylı kurulum ve çalışma adımları daha sonra backend, mobile ve admin tamamlandıkça doldurulacaktır.
